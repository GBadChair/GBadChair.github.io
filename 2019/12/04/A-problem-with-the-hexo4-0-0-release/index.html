A problem with the hexo4.0.0 release

2019-12-04

question:

hexo4.0.0会给一些a标签添加target=”_blank”，而原本a标签是默认本页打开的，导致打开a标签会跳出一个页面about:blank#blocked，
在github社区可以看到这个问题的讨论。

针对这个问题：

  • 方案一：hexo退回到3.9.0；
  • 方案二：修改hexo根目录的_config.yml配置文件；

    external_link:
      enable: false
